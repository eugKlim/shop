## The project is written in:

- React 19, TypeScript
- Redux Toolkit + RTK Query, React Hook Form, Zod
- Tailwind
- React Router 7, @splidejs/react-splide

<br>

1. Responsive design for small devices, burger menu.
2. Skeleton loading.

6. Themes: light/dark.
7. Product page with detailed description and image slider.

9. Google Maps integration.

# Installing and running the project:

1. Clone the repository:

```ruby
git clone https://github.com/eugKlim/shop.git
```

2. Install dependencies:

```ruby
npm install
```

3. Launch a project:

```ruby
npm run dev
```
